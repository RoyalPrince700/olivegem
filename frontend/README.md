# Olive Gem Global Company Ltd - Frontend

A high-end, corporate-grade frontend platform for Olive Gem Global Company Ltd, specializing in procurement services for the Oil & Gas, Agriculture, and Mining sectors. Inspired by global industry leaders, the platform features an immersive, responsive, and performance-optimized UI.

## ğŸ› ï¸ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. Navigate to the frontend directory:
   ```powershell
   cd frontend
   ```

2. Install dependencies:
   ```powershell
   npm install
   ```

3. Start the development server:
   ```powershell
   npm run dev
   ```

4. Build for production:
   ```powershell
   npm run build
   ```
